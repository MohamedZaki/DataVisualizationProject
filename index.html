Réalisé par:

  • Mohamed Zaki Chellali

Description

Contexte: Visualiser les différents clusters obtenu après application d’algorithmes de clustering, ainsi que les caractéristiques de ces établissements.

Problématique: Quelles sont les caractéristiques communes entre les banques de chaque cluster ? Les variables qui caractérisent le mieux chaque cluster ? La qualité de la classification est-elle optimale ?

Visulalisations réalisées:

Parallel Coordinates des etablissements

Small Multiples des clusters

Radar chart des clusters
